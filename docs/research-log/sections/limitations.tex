\subsection{System-Level Limitations}

Beyond perception and control, several system-level constraints limit overall capability:
\begin{itemize}
\item \textbf{Latency}: Python-based MPC optimization may exceed $50~\mathrm{ms}$ in worst-case scenarios.
\item \textbf{Static UAV assumption}: The system assumes a hovering aerial platform, with no coordinated aerial--ground motion.
\item \textbf{No obstacle avoidance}: Path-following logic assumes a clear environment without dynamic or static obstacles.
\end{itemize}
